# .workflow

Temporary HALO checkpoints can live here during active work.

Rules:

- Do not make this the only copy of a required decision.
- Durable phase outputs belong under `docs/agent_tasks/active/` or `docs/reports/`.
- Real validation artifacts belong under `test_artifacts/`.
- In real projects, add `.workflow/` to `.gitignore` unless checkpoints are
  intentionally versioned.
